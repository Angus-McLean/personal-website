Cyphor

Platform Independent End-To-End Encryption App

Summary
         Cyphor is an encryption app that enables people to communicate on their favourite websites in complete privacy. Built as a chrome extension, cyphor would seamlessly sit ontop of any social media, email or instant messaging website and encrypt data going in to the site (input boxes) and decrypt data (text elements) coming out. Using state-of-the-art encryption and web XSS policy cyphor can provably provide end-to-end encryption unbreakable to eaves-droppers, the user's platform and Cyphor itself.

Architecture Overview

  • Multi-Context Browser Extension with AngularJS UI
  • NodeJS and MongoDB RESTful backend
  • Docker and Docker-compose infrastructure
  • Socket.io realtime communciation
  • Signal Protocol, AES and ECC encryption schemes

WiFly

Flying a Quadcopter at Infinite Range

Summary
         WiFly was a 24-hr hackathon project built at RoboHacks where I built a system to fly a quadcopter at infinite range by using the cellphone network. I was able to use the RPi with a LTE module to relay commands between the WiFly.tech webserver that was hosting the command interface and the onboard flight computer (ArduCopter) running on the arduino. Using WebSockets to transmit MAVLink commands and telemetry data between web clients, the web server and the drone; users could fly the quadcopter in real-time due to surprisinly low-latency (20-100ms). The project ended up winning the Amazon Web Services prize at the hackathon and was voted one of the top hacks!

Architecture Overview

  • Raspberry Pi with LTE & Arduino with ArdoCopter
  • Socket.io realtime communciation
  • MAVLink protocol and ArduCopter telemetry communication
  • NodeJS RESTful backend
  • AngularJS front-end

DreamTeam MCT

Monitoring and Controlling a LEGO EV3 in realtime

Summary
         The Lego EV3 soccer playing robot was built with a team of 5 members for my Design Principles and Methods class at McGill. After having completed the requirements for the robot I decided to increase useability and flexibility of the robot by building an interface to monitor and control it. Using TCP/IP over Bluetooth (PAN) I was able to establish message queues and exchange JSON messages between the 2 onboard computers on the EV3 and a local socket server running on my laptop. A set of message types and parameters were then established to communicate all telemetry data and all relevant commands.

Architecture Overview

  • 2 Lego EV3 MindStorm Kits
  • Java Threads and Message Queues
  • NodeJS, Socket.io and MongoDB backend
  • Socket.io realtime communciation
  • TCP/IP over Bluetooth PAN network

Distracted Driving

Detection and classification of distracted drivers

Summary
         Using a deep sequential neural network of 2D convolutional followed by 2 fully connected perceptron layers to fine tune I was able to train a network to classify distracted drivers. The model was then trained on a labeled data set made openly available by State Farm on Kaggle and was trained on AWS GPU servers to classify future images. Then a Progressive Web App would take pictures at an interval and send them back to the server to analyze, warning beeps are triggered if the driver was seen to be driving dangerously.

Architecture Overview

  • Machine Learning built with Python and Keras
  • NodeJS RESTful backend
  • Python and Jupyter Notebook
  • Progressive Web App and UserMedia Api

Soothe

Browse the web with peace of mind

Summary
         Soothe is a Chrome extension that was built for the Carleton University hackathon and won second place overall and the HackHarrasment prize. It uses sentiment analysis and users' contributions to analyse and filter pages of their malicious content. Our main goal is to prevent online harassment and help users avoid seeing triggers while they browse the web. We strongly believe that no one online should have to worry about this.
